## Experimental Set Up

- Develop a Transformer Model that can automatically classify each bit of a CAN Message (64 bits) with a data type (counter/checksum/sensor/flag bit). 

- Manually decode CAN message IDs for Subaru and Toyota vehicles first, split decoded message IDs in to train and test datasets (CANdid message DB from 6 vehicles). Use opensource OBD reference work to help decode most messages.

- Train and test several transformer architures including different bit temporal input sizes (each bit over a sliding window 50,100,200, or 500 messages), CNN layers, attention methods (cross attention), positional encoding, firstly traning on each vehicle model separately.

- Evaluation: Evaluate with the ground truth for each message using the decoded CAN message IDs. Also test generalisation of trained models on decoded CAN message IDs different vehicles from the same OEM and different OEMs.

- Train and test similar transformer models but construct a training dataset from with CAN messages from different vehciles and OEM (in the same dateset). Evaluate how this combined transformer performs compared to the models trained separately on each vehicle.
